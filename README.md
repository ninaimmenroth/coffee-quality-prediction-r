# coffee-quality-prediction-r
This project explores the Coffee Quality Institute (CQI) Coffee Quality Database to identify patterns and factors that influence coffee cup scores. Using the programming language R, machine learning methods are implemented to predict coffee quality based on important features.

(Note: The dataset is not included in this repository; please download it directly from [Kaggle](https://www.kaggle.com/datasets/volpatto/coffee-quality-database-from-cqi/data).)
